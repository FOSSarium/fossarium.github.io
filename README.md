# 🌱 Fossarium

**Fossarium** is an **ever-growing library of Free and Open Source Software (FOSS)**.

The goal of Fossarium is simple:<br>
Create a place where people can **discover, archive, and share powerful open-source tools** without limits.

Instead of scattered repositories, lists, and websites, Fossarium aims to become a **central knowledge ecosystem for FOSS**.

---

## 📚 What Fossarium Is

Fossarium is a **curated and community-driven library** of:

* 🧰 Developer tools
* 🌐 Open-source websites
* 🤖 AI tools
* 📱 Apps
* 🎨 Creative software
* 📖 Educational resources
* ⚙️ Utilities and system tools

Everything inside Fossarium is **Free and Open Source**.

---

## 🌍 Vision

Fossarium aims to become:

> **The ultimate ever-growing library of open-source innovation.**
