Getting Started

  • Add Players with name and optional rating
  • Generate Pairings for each round using Swiss system
  • Enter Results (1 = win, 0.5 = draw, 0 = loss)
  • View Standings with automatic tie-break calculations

Pairing Rules

  • ✓ No rematches between same players
  • ✓ No player receives Bye twice
  • ✓ No player gets same color 3 times in a row
  • ✓ Color balance maintained (max ±2 difference)

Tie-Breaks (in order)

  • Buchholz: Sum of all opponents' scores
  • Buchholz Cut 1: Minus lowest opponent score
  • Sonneborn-Berger: Weighted by opponent strength
  • Progressive: Cumulative score per round
